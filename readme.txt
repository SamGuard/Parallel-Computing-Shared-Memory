Compile main.c as nomral with -lpthread flag
To run type the command:
./main _threads _width _height _precision _is_validate
replace _NAME with the values you want to run with. 
Set _is_validate to 0 if you want to just run normaly, 1 if you want it to print the data.
Typical values would be: 
./main 8 512 512 0.000001 0
